# 🍔 Food Delivery Data Analysis

## 📌 Project Overview
This project analyzes food delivery data to uncover insights related to revenue, customer behavior, and operational efficiency.

---

## 🎯 Business Problem
The goal is to identify key revenue drivers, analyze delivery performance, and improve customer satisfaction.

---

## 🛠 Tools Used
- Excel
- Power Query
- Pivot Tables
- Data Visualization

---

## 🔄 Process
1. Data Cleaning using Power Query  
2. Data Transformation  
3. Pivot Table Analysis  
4. Dashboard Creation  

---

## 📈 Key Insights
- Top cuisines drive majority of revenue  
- Delivery time impacts ratings  
- High-value customers contribute significantly  
- Peak demand observed on specific days
